About the Project :@Also see snapshots folder

This is a simple web application to implement the spring security.
The web uses spring security for login and authentication purpose.
If you are logged in already the you click on already logged in link 
and you will be redirected to your account.

For other details of the project- @see snapshots folder

Details-

Java - 1.8
For Login page - JSP
Signup Validation- JavaScript.
Database- MySql.
Hibernate Core- 4.0.3
Spring Core - 4.0.3
Spring security Core - 4.0.3
